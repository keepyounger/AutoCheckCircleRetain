# AutoCheckCircleRetain
自动检测项目中的循环引用
Checking retain circle automatically.
#why(为什么有这个)
Annoying retain circle.
烦人的循环引用。
#how to use（怎么用）
Download this project add drag the class dir to your project.
Don't need any code.
下载工程并把class文件夹的内容拖到你的工程。无需写任何代码。
#note
It works in DEBUG only.
默认只在DEBUG模式有效。
